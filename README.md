# BreastCancerDiagnosis

Breast Cancer Prediction is a data science project to predict breast cancer. The machine learning model is trained on huggingface data set (hf://datasets/scikit-learn/breast-cancer-wisconsin/breast_cancer.csv)

Models under consideration:
Logisitic Regression
XGBoost
SVC
ADA Boost classifier

Best model performance: Logistic regression
